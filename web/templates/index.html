AI 评分配置

配置 OpenAI 或 Anthropic 兼容 API，让模型为每位角色给出扮演分与影响分。配置仅保存在本地浏览器。

OpenAI 兼容: gpt-4o / DeepSeek / Moonshot / 通义 / ollama 等。Anthropic: 官方 Claude API 或第三方代理（不走 OpenAI 协议）。
兼容地址，如 OpenAI、DeepSeek、Moonshot、本地 ollama (http://localhost:11434/v1) 等。
保存在浏览器 localStorage；拉取模型与评分时会发送到本服务端，由服务端转发给模型 API。
点击「拉取列表」从 base_url + api_key 自动获取可用模型；点击 ▼ 查看全部，输入框中输入可过滤；也可手填。
高级参数（采样与生成）
单次响应上限。思考型模型（Claude 4.x、o-series）建议 8192-16384。
0=最稳定，1=平衡，1.5+=更发散。
送给模型的 IC 发言条数。
OOC 发言条数；可设 0。
送给模型的样本总字符上限（约 4-5 字 ≈ 1 token）。Claude 1M 上下文可设 100000+；GPT-4o 设 30000；ollama 看模型。
